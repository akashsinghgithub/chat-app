# MERN Chat Application

A real-time chat application built using the MERN (MongoDB, Express, React, Node.js) stack. This application supports user authentication, real-time messaging, and user profile management.

## Features

- User Authentication (Sign Up, Login)
- Real-time messaging with WebSockets
- User profile management
- Responsive UI with Chakra UI
- JWT authentication
- Error handling and validation

## Technologies Used

- **Frontend:**
  - React
  - React Router DOM
  - Chakra UI
  - Axios
- **Backend:**
  - Node.js
  - Express.js
  - MongoDB
  - Mongoose
  - JWT (JSON Web Tokens)
  - Bcrypt.js for password hashing
- **Other:**
  - WebSocket (Socket.io)
  - Cloudinary (for image uploads)
  - Vercel (for serverless deployment)

## Installation

To run this project locally, follow these steps:

1. **Clone the repository:**

   ```bash
  git clone https://github.com/akashsinghgithub/chat-app.git
   cd chat-app
   ```
## run the app
-Go-To root directory

` npm install`

### create env file
- Add PORT 5000
- Add your MONGO_URI
-Add JWT_SECRET

`npm start` To start the server.

`cd .\frontend`

`npm instal --legacy-peer-deps`

`npm start `
